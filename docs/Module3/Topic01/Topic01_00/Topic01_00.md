# Topic 1 - Python 数据分析环境搭建

在本章中，我们将学习如何搭建 Python 数据分析这个模块中所需的开发环境。我们将使用 Anaconda 这个集成环境来管理 Python 及其相关的库和工具。

- **Topic 1.1 - 下载 Anaconda 并创建 Conda 环境**：学习如何下载和安装 Anaconda，并创建一个新的 Conda 环境来管理项目所需的库和工具。
- **Topic 1.2 - Anaconda 中的其他功能（补充）**：了解 Anaconda Navigator 中集成的其他常用数据科学工具及其特点。
- **Topic 1.3 - 使用命令行管理 Conda 环境（补充）**：了解如何使用 Conda 的命令行工具来创建和管理环境以及安装包。

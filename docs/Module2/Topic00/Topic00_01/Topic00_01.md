# Python 面向对象 - 课程简介

面向对象编程是一种现代编程的范式，主要使用封装类和对象的方式来组织代码。相对于我们之前学习的面向过程编程，面向对象编程更符合人类的思维习惯，更易于理解和维护复杂的软件系统。在商业分析和数据科学领域，很多代码框架都是基于面向对象编程设计的，因此掌握面向对象编程的基本概念和技术对于商业分析专业的同学们非常重要。

本模块我们将系统地介绍 Python 面向对象编程的基本概念和技术，必须掌握的基础内容包括类和对象的基本概念、类的继承与多态等，其余内容为面向对象软件开发的进阶内容。对于商业分析专业来说，只需掌握面向对象编程的基础内容即可，这部分内容所涉及的知识点是之后后续数据分析与建模的基础。而面向对象编程的进阶内容使用频率较低，我们将以了解为主，感兴趣的同学可以进一步深入学习。

# Topic 2 - Python 面向对象进阶（补充）

本章我们将介绍 Python 面向对象编程中的一些进阶概念和技术，对于商业分析专业来说实际使用频率较低，所有知识点以了解为主。

- **Topic 2.1 - 类与类之间的关联与组合关系**：介绍类与类之间，除了继承之外，存在的两种主要关系：关联关系和组合关系。
- **Topic 2.2 - 类继承的进阶用法**：介绍较为复杂的多层继承与多重继承的用法，并学习使用 MRO 来查询类的继承顺序。
- **Topic 2.3 - 类对象**：了解 Python 中类对象的概念，学习如何创建类属性和类方法，并了解一些特殊的类。
- **Topic 2.4 - 单例**：了解单例的概念，并学习如何在 Python 中通过重写 `__new__` 方法实现单例模式。

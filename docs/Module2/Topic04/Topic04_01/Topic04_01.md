# Topic 4.1 - 字符串的基本概念

**字符串**是一段有序的文字序列，可以简单地把字符串理解为 Python 中的文字。

我们之前练习过的 `print("Hello World!")` 里面的 `"Hello World!"` 就是一个字符串。

## 1. 字符串的定义

Python 中的字符串包括**单行字符串**和**多行字符串**。

### (1) 单行字符串

单行字符串可以使用**一对双引号**（`"..."`）或者**一对单引号**（`'...'`）定义：

- 使用**一对双引号定义**的单行字符串中，双引号是表示字符串开始结束的特殊符号，要想在文字中加入双引号，需要使用 `\"`
- 使用**一对单引号定义**的单行字符串中，单引号是表示字符串开始结束的特殊符号，要想在文字中加入单引号，需要使用 `\'`

```python
a1 = "Hello World!"
b1 = 'Hello World!'

print(a1)
print(b1)
print(type(a1))
print(type(b1))

a2 = "Hello '\" World!"
b2 = 'Hello \'" World!'

print(a2)
print(b2)
```

```text
Hello World!
Hello World!
<class 'str'>
<class 'str'>
Hello '" World!
Hello '" World!
```

如果我们在引号中间不加入任何内容：`""` 或 `''`，我们就定义了一个**空字符串**：

- (1) 空字符串不是 `None` 值
- (2) 空字符串不是空格，空格是 `" "` 或 `' '`

```python
s = ""
print(s)
print(type(s))
```

```text

<class 'str'>
```

### (2) 多行字符串

多行字符串我们之前其实见过，就是多行注释，多行字符串中是支持换行的。

多行字符串可以使用**三对双引号**（`"""..."""`）或者**三对单引号**（`'''...'''`）定义：

- 使用**三对双引号定义**的多行字符串的文字内容中，不可以出现连续的三个双引号，要使用 `\"`，出现一个或连续两个双引号是可以的
- 使用**三对单引号定义**的多行字符串的文字内容中，不可以出现连续的三个单引号，要使用 `\'`，出现一个或连续两个单引号是可以的

```python
c1 = """今天天气真好呀
我们一起去玩耍吧
叫上小伙伴们一起"""

d1 = '''今天天气真好呀
我们一起去玩耍吧
叫上小伙伴们一起'''

print(c1)
print()
print(d1)
print()

c2 = """今天天气真好呀
我们一起去""玩耍吧
叫上小伙伴们一起"""

d2 = '''今天天气真好呀
我们一起去''玩耍吧
叫上小伙伴们一起'''

print(c2)
print()
print(d2)
print()

c3 = """今天天气真好呀
我们一起去""\"玩耍吧
叫上小伙伴们一起"""

d3 = '''今天天气真好呀
我们一起去''\'玩耍吧
叫上小伙伴们一起'''

print(c3)
print()
print(d3)
print()
```

```text
今天天气真好呀
我们一起去玩耍吧
叫上小伙伴们一起

今天天气真好呀
我们一起去玩耍吧
叫上小伙伴们一起

今天天气真好呀
我们一起去""玩耍吧
叫上小伙伴们一起

今天天气真好呀
我们一起去''玩耍吧
叫上小伙伴们一起

今天天气真好呀
我们一起去"""玩耍吧
叫上小伙伴们一起

今天天气真好呀
我们一起去'''玩耍吧
叫上小伙伴们一起

```

使用多行字符串时，要额外注意两点：

- (1) 多行字符串在不换行的时候，也是可以定义单行字符串的
- (2) 多行字符串中，如果把开头的三个引号单独放在第一行，开头就会有一个空行；同样，把结束的三个引号单独放在最后一行，结尾就会有一个空行

```python
e = """Hello Python!"""

print(e)

f = """
今天天气真好呀
我们一起去玩耍吧
叫上小伙伴们一起
"""

print(f)
```

```text
Hello Python!

今天天气真好呀
我们一起去玩耍吧
叫上小伙伴们一起

```

## 2. 字符串中的特殊字符

### (1) 转义字符

在字符串里有些字符不能直接写，需要用 `\` 来转义，常见转义字符有：

| 写法   | 含义         |
| ---- | ---------- |
| `\'` | 单引号        |
| `\"` | 双引号        |
| `\n` | 换行         |
| `\t` | 水平制表符   |
| `\v` | 垂直制表符       |
| `\r` | 回车（覆盖行首内容） |
| `\\` | 反斜杠        |

- 单引号和双引号我们之前见过了
- 换行符 `\n` 表示另起一行

```python
print("Hello\nWorld\n!")
```

```text
Hello
World
!
```

- 水平制表符 `\t` 是在控制台中打印多行文字时，让文字在水平方向上对齐

```python
print("Hello\nWorld\n!")
```

```text
Hello
World
!
```

- 回车 `\r` 和大家在 Word 里用的回车不一样，这里这个回车的意思是让光标回到行首，并覆盖行首的内容

```python
print("12345\rABCDE")
```

```text
ABCDE
```

- 可以看到 Python 中，反斜杠这个符号是有特殊意义的，因此，如果要在字符串中加入反斜杠，则需要两个反斜杠 `\\` 来表示

```python
print("A\\B\\C\\D\\E")
print("\\n")
```

```text
A\B\C\D\E
\n
```

### (2) 原始字符

通过上述例子可以看到，`\` 在 Python 字符串中是有特殊含义的，使用起来也比较麻烦。

要想不对 `\` 转义，一个方法是使用 Python 中的原始字符串：在字符串前面加一个 `r`：

```python
print(r"C:\new_folder\test.txt")
```

```text
C:\new_folder\test.txt
```

## 3. 计算机中文字的编码方式（补充）

由于计算机只认识 0 和 1 这两个二进制数字，要想让计算机认识文字，就要使用编码方式，来将数字和文字联系起来

### (1) ASCII

ASCII（American Standard Code for Information Interchange，美国信息交换标准代码）是最早的编码标准化方案。

ASCII 使用一个8位二进制数表示一个字符，因此 ASCII 一共可以表示 `2 ** 8 = 256` 个字符，包括：

- 大小写英文字母
- 阿拉伯数字
- 英文标点符号
- 控制符号（回车、换行、空格等）

可以看到，ASCII 可以表示的文字内容非常有限，因此不适用于现代计算机中


### (2) Unicode

现代计算机的基本需求就是可以表达世界上所有的文字和符号，这就要使用到 Unicode 了：

- Unicode 是一种编号体系，不规定具体存储方式，UTF-8 这个体系中最常用的一种编码方案
- UTF-8 基本覆盖了世界上所有的文字、标点、特殊符号、甚至emoji
- **大家之后编程处理文字时如果出现乱码，很可能是编码方式没有设置为 UTF-8**

UTF-8 使用 1-4 个字节来表示一个字符，通常越复杂越不常用的字符，所需的字符数就越多

### (3) 查看字符对应的编码值

Python 中，我们可以查看单个字符的对应的编码值，使用的函数是 `ord()`：

- 如果字符在 ASCII 码中，则返回其对应的 ASCII 码值
- 如果字符不在 ASCII 码中，则返回其对应的 Unicode 码值

```python
print(ord("A"))  # ASCII
print(ord("w"))  # ASCII
print(ord("1"))  # ASCII
print(ord("!"))  # ASCII
print(ord("\n")) # ASCII
print(ord("财"))  # Unicode
print(ord("😊")) # Unicode
```

```text
65
119
49
33
10
36130
128522
```

注意，`ord()` 函数只支持转换单个字符，不支持转换整个字符串，其实整个字符串对应的编码值，就是把单个字符的编码值串起来即可

```python
print(ord("abc"))
```

```text
TypeError: ord() expected a character, but string of length 3 found
```

如果想把编码转回对应的字符，可以使用 `chr()` 函数

```python
print(chr(65))
print(chr(119))
print(chr(49))
print(chr(33))
print(chr(10))
print(chr(36130))
print(chr(128522))
```

```text
A
w
1
!


财
😊
```

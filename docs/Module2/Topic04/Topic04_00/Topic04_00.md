# Topic 4 - 勇士与地下城游戏项目 - Python 面向对象结课项目

本节我们将开发一个简单的“勇士与地下城”游戏项目，使用 Python 的面向对象编程（OOP）概念来实现游戏的基本功能。

- **Topic 4.1 - 游戏项目介绍**：分析游戏的基本需求，设计游戏中的主要类和对象，并且实现游戏的主程序框架与辅助功能。
- **Topic 4.2 - 角色类与道具类的设计与实现**：设计并实现游戏中的角色类（包括子类玩家与怪物）和道具类（包括子类剑、盾、与血瓶），并且实现游戏中的战斗逻辑。
- **Topic 4.3 - 地图类的设计与实现**：设计并实现地图类，重点关注地图中的移动玩家方法，并且实现判断战斗触发与道具拾取的逻辑。
- **Topic 4.4 - 整合各个游戏功能到主程序**：将前面实现的各个类和功能整合到主程序中，完成游戏的整体流程。
